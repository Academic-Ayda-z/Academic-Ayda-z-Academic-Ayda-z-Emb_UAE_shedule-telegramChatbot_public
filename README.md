# Academic-Ayda-z-Academic-Ayda-z-Emb_UAE_shedule-telegramChatbot_public


In 2024, I needed to schedule my US visa appointment and make it scalable since I had a direct, fully funded PhD position admission to UConn (biomedical engineering).

This is about getting an appointment from the US embassy. It navigates the website, inputs the user's data and login to the account and looks for the open dates frequently. I also implemented a chatbot in Telegram to be able to scale it. It works and gets the dates and even selects the date. At the last stage of coforming, it fails to do it.
